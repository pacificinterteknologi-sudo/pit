# 🚀 UPLOAD COMMANDS - RUN SEKARANG!

## 📋 **Step-by-Step Upload Commands**

### 🎯 **Step 1: Navigate to Project Folder**
```bash
cd "d:/my Project/pacific_website"
```

### 🎯 **Step 2: Initialize Git Repository**
```bash
git init
```

### 🎯 **Step 3: Add Essential Files**
```bash
# Add core website files
git add index.html
git add logo.png
git add style.css
git add slider.js

# Add Firebase configuration
git add firebase.json
git add .firebaserc

# Add public folder
git add public/
git add public/index.html
git add public/logo.png

# Add documentation
git add DEPLOYMENT_GUIDE.md
git add EMAILJS_SETUP.md
git add GITHUB_INSTALL_GUIDE.md
git add UPLOAD_COMMANDS.md
git add .gitignore
```

### 🎯 **Step 4: First Commit**
```bash
git commit -m "Initial commit: Pacific Inter Teknologi Website with WhatsApp integration and professional navigation"
```

### 🎯 **Step 5: Add GitHub Repository**
```bash
git remote add origin https://github.com/USERNAME/pacific-website.git
# Ganti USERNAME dengan GitHub username Anda
```

### 🎯 **Step 6: Push to GitHub**
```bash
git push -u origin main
```

## 🚨 **Troubleshooting**

### ❌ **Common Issues:**
```bash
🔧 "git not recognized" → Install Git dulu
🔧 "Permission denied" → Check GitHub credentials
🔧 "Authentication failed" → Gunakan Personal Access Token
🔧 "Push rejected" → git pull origin main dulu
```

### ✅ **Solutions:**
```bash
🔧 Setup credentials:
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

🔧 Force push (if needed):
git push -u origin main --force
```
